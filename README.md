# Renewable Energy Monitoring System

## 📌 Description
This project is a SQL-based system designed to monitor renewable energy generation across different sources and locations.

## ⚙️ Features
- Tracks Solar, Wind, Hydro energy
- Stores production data by location
- Uses SQL queries for analysis
- Supports filtering, grouping, and sorting

## 🛠 Tech Stack
- SQL (MySQL)

## 📊 SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOIN
